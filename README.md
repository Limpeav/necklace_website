# Lunelle Jewelry

Frontend-only necklace catalog for a premium jewelry brand.

## Stack

- Frontend: React, Vite, Tailwind CSS
- Data: Local client-side product catalog sourced from `client/src/assets/product_image`

## Features

- Homepage with hero, featured products, promo banner, testimonials, and newsletter
- Shop page with search, category, material, color, price filters, and sorting
- Product details with gallery, materials, color and length selection, reviews, wishlist, and related products
- Static product gallery using local images from `client/src/assets/product_image`

## Local Setup

1. Install dependencies:

```bash
npm install
```

2. Start the site:

```bash
npm run dev
```

## Default URL

- Frontend: `http://localhost:5173`
